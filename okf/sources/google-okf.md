# Google OKF 来源

Google Cloud 将 OKF 描述为一种使用 Markdown 和结构化元数据交换 Agent 知识的标准格式。

## 在本项目中的用途

本项目把 OKF 作为知识条目的长期文件格式，并围绕它构建本地校验、图谱导出和可视化浏览能力。
