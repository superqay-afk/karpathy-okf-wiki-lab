# 知识整理

知识整理是把原始信息持续转化为稳定、可连接、可复用知识的过程。它不只是“保存资料”，更强调来源、结构、链接、复查和后续可行动性。

## 质量检查

- 必要元数据完整存在。
- 外部来源链接被保留。
- 内部 Markdown 链接可以正常解析。
- 标签足够具体，便于过滤和组合检索。
- 孤立页面会被暴露出来，进入后续评审。
- 重要整理决策记录在[整理日志](../log.md)中。
